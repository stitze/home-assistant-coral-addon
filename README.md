# Home Assistant Coral Add-on

A base add-on to enable Google Coral Edge TPU support in Home Assistant.

## Features
- Installs Edge TPU Runtime
- Installs PyCoral
- Hardware passthrough for USB/PCIe

## Installation
1. Add this repository to your Home Assistant Supervisor.
2. Install the "Home Assistant Coral Add-on".
3. Start the add-on and check the logs.
